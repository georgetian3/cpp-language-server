\section{概述}

此项目实现了 C++ 的词法分析器、语法分析器、以及代码高亮与提示程序。此项目的大部分逻辑实现于 Python 3，其中关于编译的算法与逻辑使用了 \href{https://github.com/dabeaz/ply}{PLY}。实现过程中主要参考的资料是 C++20 标准 \href{https://isocpp.org/files/papers/N4860.pdf}{N4860} 版本。
